# Neobanker前后端接口规范文档
---
## 概述  

本文档旨在定义前后端交互的接口规范，以确保开发过程中前后端能够高效、准确地进行数据交换。接口规范包括请求方法、请求参数、响应体结构、错误处理等方面的规定。  
## AccountController
#### login

**

**请求URL**

/account/login `POST` 


**返回结果**

```json
{
	"email":"string",
	"userName":"string",

	"role":"string",
	"status":"string"
}
```
